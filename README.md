# 🛡 Scam Detector

An AI-powered web application that detects spam and suspicious scam messages using Machine Learning.

Users can paste suspicious messages into the app and instantly receive predictions indicating whether the message is safe or potentially fraudulent.

---

# 🚀 Features

- 🔍 Spam & scam message detection
- 🤖 Machine Learning text classification
- ⚡ Real-time predictions
- 🎨 Clean React frontend UI
- 🔗 Flask backend API
- 🆓 Free trial scan system
- 🚨 Scam/spam warning alerts

---

# 🧠 How It Works

The application uses a trained Machine Learning model to analyze text patterns and classify messages as:

- Spam
- Safe (Ham)
